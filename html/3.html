Mozilla Foundation 's prototype-based programming language , which corresponds to a scripting language . All web browsers have a built-in interpreter, except for special purposes . Today , along with HTML and CSS , it is one of the elements that compose the web. While HTML is responsible for the basic structure of a web page and CSS is responsible for design, JavaScript is responsible for operating the web page on the client side.
